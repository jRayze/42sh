NAME
       color


SYNOPSIS
       color [arg]


DESCRIPTION
       The  color utility put the color on or off.  If there is no argument or
       if the argument is "on", the color is put on.  Otherwise, the  color is
       put off.


EXIT STATUS
       This utility returns 0 on success and >0 if error occurs.
